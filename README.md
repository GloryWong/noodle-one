# templates

Apply part templates.

### Install

```bash
pnpm add @gloxy/templates
```
### Usage

```typescript
import { applyPartTemplates } from '@gloxy/templates'

applyPartTemplates('vscode', { force: true, merge: true })
```

### CLI

```bash
pnpm add -g @gloxy/templates
```
